# 📘 StudyNotes

Sistema de anotações desenvolvido com ASP.NET Core MVC.

## 🚀 Funcionalidades
- Criar notas
- Editar notas
- Excluir notas
- Buscar notas

## 🛠️ Tecnologias
- C#
- ASP.NET Core
- Entity Framework Core
- SQL Server

## 💻 Como rodar o projeto

1. Clone o repositório:
git clone https://github.com/seu-usuario/StudyNotes-App.git

2. Execute:
Add-Migration InitialCreate
Update-Database

3. Rode o projeto
